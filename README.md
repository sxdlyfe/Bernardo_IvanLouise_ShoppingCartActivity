# Bernardo, Ivan Louise S. | BSIT 1-2 Computer Programming
A shopping cart system for an auto parts store. Products include headlights, taillights, from car models such as Vios, Innova, Fortuner and more car brands and car models. 

## App Features
- Browse 20 car parts and auto parts items from brands like Toyota, Honda, Mitsubishi and other misc. car parts.
- Add items to the cart by choosing ID and entering quantity.
- Input validation is enforced (typing characthers instead of integer will reset the program).
- Duplicate item mergin in cart.
- 10% discount applied if cart is totaling or more than ₱5,000.
- Receipt generation including item name, quantity, and total price.
- Update stock after checkout.
